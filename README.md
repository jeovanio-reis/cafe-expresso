#  Café Expresso

##  Descrição
O Café Expresso é um sistema de autoatendimento para uma cafeteria de pequeno porte.

##  Objetivo
- Reduzir filas
- Agilizar pedidos
- Diminuir erros

##  Funcionalidades
- Cadastro de produtos
- Adicionar itens ao pedido
- Calcular valor total
- Gerenciar status do pedido:
  - Pendente
  - Pago
  - Em preparo
  - Finalizado

##  Estrutura (Classes)
- Produto
- ItemPedido
- Pedido
